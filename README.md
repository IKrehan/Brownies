# Brownies E-Commerce

Um e-commerce desenvolvido com Python(Flask), HTML e CSS.

[browniesproject.herokuapp.com](http://browniesproject.herokuapp.com)



## Servidor Local
Para rodar localmente execute os comandos seguintes:

```bash
pip install -r requirements.txt
```

```bash
set FLASK_APP=Site
set FLASK_DEBUG=1
flask run
```
